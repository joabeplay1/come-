# 📿 Dominó Brasileiro Online — Suíte Premium

Um jogo de Dominó completo desenvolvido em HTML5 moderno, CSS estruturado (Glassmorphism feltro design) e Firebase Firestore para sincronização multiplayer em tempo real de altíssima performance.

## 🚀 Funcionalidades Inclusas neste Deploy
- **Multiplayer Online Nativo:** Conecte dois jogadores instantaneamente por sistema de código de sala.
- **Regras Oficiais do Dominó Brasileiro:** Distribuição exata de 28 peças, compra de peças direto no "carro" e validação rígida de lados compatíveis na mesa.
- **Sistema Contábil de Apostas:** Definição prévia de moedas antes do início de cada partida.
- **Chat Sincronizado:** Converse com seu oponente diretamente através do painel lateral integrado.
- **Efeitos Sonoros Integrados:** Respostas auditivas de simulação física de peças gerados por oscilação por Web Audio API.

## 🛠️ Como Configurar suas Credenciais
1. Acesse o [Firebase Console](https://console.firebase.google.com/).
2. Crie um projeto e ative o **Firestore Database**.
3. Obtenha os dados do objeto `firebaseConfig` e preencha o arquivo `firebase.js`.
4. Faça o commit para a branch `main` no seu repositório e ative as **GitHub Pages**.
